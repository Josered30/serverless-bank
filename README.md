# Serverless Bank

Este proyecto simula transacciones para el abono de fondos a una cuenta bancaria.

## Tecnologias

### Rust
- Cargo Lambda (https://www.cargo-lambda.info/)

### AWS Cloud
- AWS DynamoDB
- AWS Lambda
- AWS Step Function
- AWS API Gateway
- AWS Cloudformation
- AWS SAM
